# TASTE Token

TASTE is a Jetton token built on the TON blockchain.

## Token Information
- Name: TASTE
- Symbol: TASTE
- Network: TON
- Standard: Jetton (TON)
- Total Supply: 25,000,000 TASTE
- Decimals: 9

## Description
TASTE is designed as a utility and discount token used within the Taste ecosystem.
It enables users to benefit from discounts, rewards, and digital value exchange
without relying on centralized exchanges.

## Key Features
- Fixed supply (no future minting)
- Jetton standard compliant
- Designed for real-world utility
- Transparent and community-driven

## Blockchain
- TON Blockchain
- Wallet compatible with TON Space and other TON wallets

## Status
This repository is public for transparency and community review.
